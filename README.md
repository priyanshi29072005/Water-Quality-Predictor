# 💧 Water Quality Predictor

## 📌 Problem
Access to safe drinking water is a major concern, especially in urban areas. It is difficult to determine whether water is safe without laboratory testing.

## 💡 Solution
This project uses Machine Learning to predict whether water is safe to drink based on chemical properties.

## ⚙️ Features Used
- pH
- Hardness
- Solids
- Chloramines
- Sulfate
- Conductivity

## 🧠 Model Used
Random Forest Classifier

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the program:
python model.py

## 📊 Output
- Predicts whether water is SAFE or NOT SAFE
- Shows model accuracy
